# Sneaker Shop

## Sovelluksen toiminnot

* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan myynti-ilmoituksia.
* Käyttäjä pystyy lisäämään kuvia ilmoitukseen.
* Käyttäjä näkee sovellukseen lisätyt ilmoitukset.
* Käyttäjä pystyy etsimään ilmoituksia hakusanalla.
* Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja ja käyttäjän lisäämät ilmoitukset.
* Käyttäjä pystyy valitsemaan ilmoitukselle yhden tai useamman luokittelun (esim. kenkien merkki, kenkien koko, kenkien kunto, vastaanottotapa (meetup tai vain lähetys)).
* Käyttäjä pystyy tarjoamaan myytävänä olevalle tuotteelle oman hinnan.
